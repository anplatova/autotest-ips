Написать кейсы, реализовать по ним автотесты:
a.добавление label к issue (создание label и issue через API)
b.удаление label у issue (создание задачи с label через API)
c.после тестов нужно удалить все label

#001
1) Создать label через API
2) Создать issue через API
3) Открыть url созданного issue
4) Кликнуть на Labels в боковом меню
5) Выбрать для issue созданный label
6) Дождаться отображения записи о привязке label
7) Открыть страницу /labels
8) Ввести в поле поиска lablels название созданного label
9) Дождаться отображения результатов поиска
Ожидаемый результат: В списке задача созданная в п2

#002
1) Создать issue с label через API 
2) Открыть url созданного issue
3) Кликнуть на Labels в боковом меню
4) Нажать Enter для снятия выбора label
5) Дождаться отображения записи об удалении label из issue
6) Открыть страницу /labels
7) Ввести в поле поиска lablels название созданного в п1 label
8) Дождаться отображения результатов поиска
Ожидаемый результат: в списке нет созданной задачи, отобрается блок без задача

#003
1) Удалить через API созданные в кейсах 001 и 002 labels по их именам
2) Открыть страницу /labels
3) Ввести в поле поиска lablels название созданного в кейсе 001 label
4) Дождаться отображения результатов поиска
5) Ввести в поле поиска lablels название созданного в кейсе 002 label
6) Дождаться отображения результатов поиска
Ожидаемый результат: в результатах поиска отображается пустой блок, без labels
