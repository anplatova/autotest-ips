Тест-кейсы по странице профиля

Предварительные шаги:
1) Открыть в браузере https://github.com/login 
2) Залогиниться со своими логином и паролем
3) Открыть https://github.com/settings/profile

#001
Редактирование поля Name
Шаги воспроизведения:
1) Дождаться загрузки поля Name
2) Ввести в поле ввода "Name" значение "test name"
3) Дождаться загрузки кнопки Update profile
4) Нажать на кнопку Update profile
5) Открыть https://github.com/anplatova
6) Дождаться загрузки имени профиля
Ожидаемый результат: На странице отображается имя пользователя "test name"

#003
Отображение email на странице профиля
Шаги воспроизведения:
1) Дождаться загрузки селекта Public email
2) Кликнуть на селект
3) Выбрать в селекте email anplatova18@gmail.com
4) Дождаться загрузки кнопки Update profile
5) Нажать на кнопку Update profile
6) Открыть https://github.com/anplatova
7) Дождаться загрузки имени email под именем профиля
Ожидаемый результат: На странице отображается email anplatova18@gmail.com

#005
Загрузка аватара поддерживаемого типа (PNG, GIF, or JPG)
Шаги воспроизведения:
1) Дождаться загрузки кнопки смены аватара Edit
2) Нажать на кнопку Edit
3) Нажать на кнопку Upload a photo…
4) Загрузить картинку в формате PNG
5) Нажать на кнопку сохранения аватара "Set new profile picture"
Ожидаемый результат: На странице появится плашка о смене аватара "Your profile picture has been updated. It may take a few minutes to update across the site."

#006
Ошибка при загрузке аватара неподдерживаемого типа
Шаги воспроизведения:
1) Дождаться загрузки кнопки смены аватара Edit
2) Нажать на кнопку Edit
3) Нажать на кнопку Upload a photo…
4) Загрузить файл неподдерживаемого типа
Ожидаемый результат: Отображается сообщение об ошибке "We only support PNG, GIF, or JPG pictures."
