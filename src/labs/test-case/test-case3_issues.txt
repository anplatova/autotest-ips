Тест-кейсы по странице issues

Предварительные шаги:
1) Открыть в браузере https://github.com/login 
2) Залогиниться со своими логином и паролем
3) Открыть https://github.com/anplatova/test-for-study/issues

#001
Создание задачи с допустимым количеством символов в названии
Шаги воспроизведения:
1) Нажать кнопку Новая задача
2) Заполнить поле названия задачи с использованием сгенерированной строки
3) Нажать кнопку Создания задачи
Ожидаемый результат: Название созданной задачи соответствует введенному названию в шаге 3

#002
Создание задачи с не допустимым количеством символов в названии
Шаги воспроизведения:
1) Нажать кнопку Новая задача
2) Заполнить поле названия задачи строкой из 1025 символов
3) Нажать кнопку Создания задачи
Ожидаемый результат: Отображается сообщение об ошибке количества символов в названии
