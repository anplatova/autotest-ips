Тест-кейсы по странице issues

Предварительные шаги:
1) Открыть в браузере https://github.com/login 
2) Залогиниться со своими логином и паролем
3) Открыть https://github.com/anplatova/test-for-study/issues

#001
Создание задачи с допустимым количеством символов в названии
Шаги воспроизведения:
1) Нажать кнопку Новая задача
2) Дождаться загрузки поля Название
3) Заполнить поле названия задачи с использованием сгенерированной строки и допустимым количеством символов
4) Дождаться загрузки кнопки Создания задачи
5) Нажать кнопку Создания задачи
6) Дождаться отображения названия созданной задачи
Ожидаемый результат: Название созданной задачи соответствует сгенерированной строке

#002

6) дождаться отображения сообщения об ошибке
Ожидаемый результат: Отображается сообщение об ошибке

#006
Возможность оставлять комментарии к задаче, если они включены
Шаги воспроизведения:
1) Нажать кнопку Новая задача
2) Заполнить поле названия задачи рандомным значением
3) Нажать кнопку Создания задачи
4) Сохранить url созданной задачи
5) Открыть https://github.com/logout
6) Нажать на кнопку разлогинации
7) Открыть https://github.com/login
8) Залогиниться с логином и паролем комментатора
9) Открыть url созданной задачи
10) Дождаться загрузки поля ввода комментария
11) В поле комментария ввести рандомную строку
12) Дождаться кликабельности кнопки сохранения комментария
13) Нажать кнопку сохранения комментария
14) Дождаться отображения текста комментария
Ожидаемый результат: В задаче отображается комментарий, который ввели в пункте 11
