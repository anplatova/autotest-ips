Тест-кейсы по странице issues

Предварительные шаги:
1) Открыть в браузере https://github.com/login 
2) Залогиниться со своими логином и паролем
3) Открыть https://github.com/anplatova/test-for-study/issues

#001
Создание задачи с допустимым количеством символов в названии
Шаги воспроизведения:
1) Нажать кнопку Новая задача
2) Дождаться загрузки поля Название
3) Заполнить поле названия задачи с использованием сгенерированной строки и допустимым количеством символов
4) Дождаться загрузки кнопки Создания задачи
5) Нажать кнопку Создания задачи
6) Дождаться отображения названия созданной задачи
Ожидаемый результат: Название созданной задачи соответствует сгенерированной строке

#002
Создание задачи с не допустимым количеством символов в названии
Шаги воспроизведения:
1) Нажать кнопку Новая задача
2) Дождаться загрузки поля Название
3) Заполнить поле названия задачи строкой из 1026 символов
4) Дождаться загрузки кнопки Создания задачи
5) Нажать кнопку Создания задачи
Ожидаемый результат: Отображается сообщение об ошибке

#003
Редактирование названия созданной задачи
Шаги воспроизведения:
1) Нажать кнопку Новая задача
2) Заполнить поле названия задачи рандомной строкой
3) Нажать кнопку создания задачи
4) Нажать на кнопку редактирования
5) Изменить навзание на рандомное значением
6) Нажать на кнопку Сохранить
Ожидаемый результат: у задачи новое навазние

#004

Шаги воспроизведения:
1) Нажать кнопку Новая задача
2) Заполнить поле названия задачи рандомным значением

#005

Шаги воспроизведения:
1) Нажать кнопку Новая задача

Ожидаемый результат: Отображается сообщение об ошибке недопустимого формата файла

#006
Возможность оставлять комментарии к задаче, если они включены
Шаги воспроизведения:
1) Нажать кнопку Новая задача
2) Заполнить поле названия задачи рандомным значением
3) Нажать кнопку Создания задачи
4) Сохранить url созданной задачи
5) Открыть боковое меню пользователя
6) Нажать кнопку разлогинации
7) Открыть https://github.com/login
8) Залогиниться с логином и паролем комментатора
9) Открыть url созданной задачи
10) В поле комментария ввести рандомную строку
11) Нажать кнопку сохранения комментария
Ожидаемый результат: В задаче отображается комментарий, который ввели в пункте 11

#009
Поиск задачи по тегу
Шаги воспроизведения:
1) Открыть https://github.com/anplatova/autotest-ips/labels
2) Нажать кнопку Новый тег
3) Заполнить поле названия тега с использованием сгенерированной строки
4) Нажать кнопку Создания тега
5) Открыть https://github.com/anplatova/autotest-ips/issues
6) Нажать кнопку Новая задача
7) Заполнить поле названия задачи с использованием строки из пункта 3
8) Нажать кнопку Создания задачи
9) Нажать кнопку выбора тегов
10) Заполнить поле поиска тега строкой из пункта 3
11) В списке найденных тегов выбрать созданный тег
12) Открыть https://github.com/anplatova/autotest-ips/labels
13) Заполнить поле поиска по тегу строкой из пункта 3
14) В списке найденных тегов выбрать созданный тег
Ожидаемый результат: Название найденной задачи соответствует строке из пункта 3
